Aide & Tutoriel

Apprenez à utiliser Aim Display et gérez vos compétitions sereinement

Bienvenue sur Aim Display

Aim Display est votre outil complet pour la gestion de compétitions de tir sportif. Que vous soyez un club ou un organisateur indépendant, cette section vous guidera à travers les fonctionnalités essentielles.

Démarrage Rapide

  • 1 Créez vos clubs : Créez tous les clubs participants à votre compétition
  • 2 Créez la compétition : Configurez les détails de votre événement sportif.
  • 3 Créez les tireurs : Enregistrez les participants de votre compétition ainsi que leurs scores.

Affichage & Rapports

  • Mode Affichage : Utilisez le mode plein écran pour projeter les résultats en direct aux participants et spectateurs de votre compétition.
  • Exports PDF : Générez des rapports PDF détaillés par tireur ou par compétition à la fin du tournoi.

Tutoriel Détaillé

Étape 1 : Préparation des Clubs

Avant de lancer votre compétition, enregistrez tous les clubs participants via le menu Nouveau club. Cela vous permettra de lier rapidement chaque tireur à son club d'origine. Les clubs sont conservés dans votre base de données pour une réutilisation future.

Étape 2 : Configuration de la Compétition

Utilisez l'onglet Nouvelle compétition pour définir le nom, la date et le type de tir. Assurez-vous de bien paramétrer les options spécifiques (nombre de séries, temps de tir) avant d'enregistrer.

Étape 3 : Inscription des Tireurs

Une fois la compétition créée, ajoutez les tireurs dans Nouveau tireur. Sélectionnez la compétition correspondante et le club du tireur. Rappel : un tireur est enregistré spécifiquement pour une compétition donnée.

Étape 4 : Déroulement et Résultats

Lancez le Mode Affichage sur un écran secondaire pour le public. Saisissez les scores au fur et à mesure. À la clôture, exportez le Rapport PDF global pour archiver le classement final.
